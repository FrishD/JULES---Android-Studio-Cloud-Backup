JULES - Android Studio Cloud Backup

JULES (JetBrains User-Level Encrypted Storage) is an IDE plugin for Android Studio and IntelliJ IDEA, developed by Shay Frishwasser. It enables developers to securely back up project files to their personal Google Drive storage.

Privacy Policy

Last Updated: March 4, 2026

1. Data Collection & Disclosure

JULES is a client-side only application. We do not operate any backend servers. We do not collect, store, transmit, or share any personal data, authentication tokens, or source code with any third parties, including the developer.

2. Google OAuth & Scopes

JULES uses Google OAuth 2.0 to authenticate users and interact with Google Drive. We request the following scopes:

https://www.googleapis.com/auth/drive.file
Used to create and manage the backup ZIP archives within your Drive.

https://www.googleapis.com/auth/drive.appdata
Used to store and sync your plugin configuration (e.g., backup intervals).

The plugin only accesses files created by JULES. It cannot read, scan, or index other files in your Google Drive.

3. Data Security (AES-256)

All project data is encrypted locally using AES-256 before being transmitted to Google Drive. The encryption keys are managed by the user within the IDE. The developer has no access to your encryption keys or your backed-up data.

4. Google Limited Use Policy

JULES's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

5. Data Retention & Deletion

Users have full control over their data. You can delete backups at any time via the plugin or the Google Drive web interface. To revoke access, visit Google Security Settings.
